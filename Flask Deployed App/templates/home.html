What This App Does
01
Leaf Image Input

Upload a photo or capture one from the camera directly in the AI Engine with a workflow built around leaf-based prediction.

02
Confidence Scores

Each prediction returns a model confidence percentage so the user can judge whether a result looks decisive or uncertain.

03
Top-3 Ranking

The result page shows the best three candidate classes rather than forcing a single opaque answer with no alternatives.

Private Dashboard

Logged-in users get a separate dashboard with their own scan history, prediction trends, and recent activity.

Workflow
How It Works
1. Capture a clean leaf image Use one visible leaf with clear disease signs and minimal background noise.
2. Run inference through the CNN model The model processes the image and scores all supported plant classes.
3. Review ranked outputs and guidance The UI shows the main prediction, alternatives, and disease-prevention guidance.
